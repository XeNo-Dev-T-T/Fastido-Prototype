# 🚖 Fastido

A simple **Python-based ride booking simulation** inspired by modern cab booking platforms. Fastido demonstrates the fundamentals of **Object-Oriented Programming (OOP)**, modular programming, and basic payment processing through a command-line interface.

---

## ✨ Features

* 👤 Driver onboarding system
* 🚗 Ride booking simulation
* 🎯 Automatic driver assignment
* 💳 Wallet-based payment system
* 📍 Distance-based fare calculation (₹8/km)
* 🚘 Vehicle-specific driver matching
* 🛡️ Input validation for smoother user experience
* 📦 Modular project structure

---

## ⚙️ How It Works

1. Launch the application.
2. Enter your name.
3. Request a ride.
4. Enter travel distance.
5. Select your preferred vehicle.
6. A matching online driver is assigned.
7. Confirm pickup.
8. Complete the ride.
9. Fare is automatically calculated.
10. Payment is deducted from the user's wallet.

---

## 💰 Fare Calculation

```
Fare = Distance × ₹8
```

Example:

```
Distance: 15 KM
Fare: ₹120
```
